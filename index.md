# 개인정보 처리방침

**최종 업데이트: 2026년 4월 19일**

SIM(이하 "앱")은 사용자의 개인정보를 소중히 여기며, 관련 법령에 따라 아래와 같이 개인정보를 처리합니다.

## 1. 수집하는 정보

### 위치 정보
- **수집 항목**: 촬영 시점의 GPS 좌표
- **수집 목적**: 사진 메타데이터(EXIF)에 촬영 위치를 저장
- **저장 방식**: 기기 내 사진 파일에만 저장되며, 외부 서버로 전송하지 않습니다
- **선택 여부**: 사용자가 위치 권한을 거부할 수 있으며, 이 경우 위치 정보 없이 촬영됩니다

### 사진 및 영상
- **수집 항목**: 전면/후면 카메라로 촬영한 사진
- **수집 목적**: 전후면 이미지 합성 및 사진 라이브러리 저장
- **저장 방식**: 기기 내 사진 라이브러리에만 저장되며, 외부 서버로 전송하지 않습니다

### 앱 설정
- **수집 항목**: 사용자 설정값 (테두리 색상, 텍스트, 포맷 등)
- **수집 목적**: 앱 재실행 시 설정 복원
- **저장 방식**: 기기 내 UserDefaults에만 저장됩니다

## 2. 수집하지 않는 정보

SIM은 다음 정보를 **수집하지 않습니다**:

- 이름, 이메일, 전화번호 등 개인 식별 정보
- 사용 패턴, 행동 분석 데이터
- 광고 식별자 (IDFA)
- 마이크 음성 데이터

## 3. 데이터 전송

SIM은 **모든 데이터를 기기 내에서만 처리**합니다. 외부 서버, 클라우드, 또는 제3자에게 어떠한 데이터도 전송하지 않습니다.

## 4. 제3자 공유

수집된 정보는 제3자에게 제공되지 않습니다.

## 5. 데이터 보관 및 삭제

- 사진: 기기 사진 라이브러리에서 직접 삭제할 수 있습니다
- 앱 설정: 앱 삭제 시 자동으로 삭제됩니다

## 6. 아동 개인정보

SIM은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 7. 변경 사항

본 개인정보 처리방침이 변경될 경우, 앱 업데이트를 통해 안내합니다.
